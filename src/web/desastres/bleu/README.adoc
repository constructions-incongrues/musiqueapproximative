= Désastre Bleu

== Description

Le désastre "bleu" applique un effet de glitch visuel sur les titres des émissions (`article h2`), créant une animation chaotique avec des distorsions de couleur et de position.

== Déclenchement

Le désastre est déclenché lorsque le titre de l'émission ou l'artiste contient "bleu" (insensible à la casse).

*Règle :*

[,yaml]
----
- query: "query.title ~ /.*bleu.*/i || query.artist ~ /.*bleu.*/i"
  recettes: [bleu]
  probability: 0.7
----

== Configuration

=== Options disponibles

Aucune option configurable pour ce désastre. Les effets visuels sont entièrement définis en CSS.

=== Exemple de configuration

[,yaml]
----
bleu:
  enabled: true
  desastre: bleu
----

== Fonctionnement

. *Chargement CSS* : Le fichier `styles.css` est chargé automatiquement
. *Animation glitch* : Une animation CSS `glitch` est appliquée sur `article h2`
. *Effet visuel* :
 ** Changements de couleur (cyan, bleu violet, blanc)
 ** Transformations (translation, rotation avec `skew`)
 ** Animation en boucle infinie de 1 seconde
. *Pseudo-éléments* : Des animations supplémentaires `glitchTop` et `glitchBotom` créent des effets de découpe (clip-path)

== Effets CSS

=== Animation principale

* *Durée* : 1s
* *Répétition* : infinie
* *Effet* : Translation de ±2-4px, rotation skew de ±12-45°
* *Couleurs* : Cyan (hsl(193, 100%, 81%)), Violet (hsl(261, 100%, 50%)), Blanc

=== Pseudo-éléments

* `div:after` : Animation de la partie basse (clip-path: 67%-100%)
* Durée : 1.5s en boucle infinie

== Fichiers

* `javascripts/script.js` : Fichier JavaScript (quasi-vide, 1 ligne)
* `stylesheets/styles.css` : Définition des animations glitch
* `README.adoc` : Cette documentation

== Compatibilité

Les animations CSS et clip-path sont supportés par tous les navigateurs modernes :

* Chrome/Edge : ✅
* Firefox : ✅
* Safari : ✅
* Opera : ✅

== Limitations

* L'effet ne cible que les éléments `article h2`
* Les couleurs et timings ne sont pas configurables
* L'animation est permanente une fois déclenchée (pas d'arrêt possible)
* Peut réduire la lisibilité du titre de l'émission
